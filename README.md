#  mini-project

> **This project is a responsive and user-friendly weather forecast webpage that provides real-time weather updates for any city worldwide. Built using HTML, CSS, and JavaScript, it fetches data from a reliable weather API to display current conditions, temperature, humidity, wind speed, and a 5-day forecast.**
